tli – Tree level improvement.¶

Calculate tree level improvements for lattice QCD observables in the spirit of [1]. Given an observable \({\mathcal O}(a/L)\), one defines the improved observable

\[{\mathcal O}_\mathrm{I}(a/L) = \frac {{\mathcal O}(a/L)} { 1 + \delta(a/L) }, \quad \delta(a/L) = \frac {{\mathcal O}(a/L) - {\mathcal O}(0)} {{\mathcal O}(0)} = \delta^{(0)}(a/L) + g_0^2 \delta^{(1)}(a/L) + \ldots\,.\]

This package contains a function to calculate \(\delta(a/L)\) for given observables. In practially all cases, only the tree-level approximation \(\mathcal O^{(0)}(a/L)\) is known analytically, so one will only be able to compute \(\delta^{(0)}\).

References

[1]G. de Divitiis et al., Universality and the approach to the continuum limit in lattice gauge theory, Nucl.Phys. B 437, 447–470, 1995, [doi: 10.1016/0550-3213(94)00019-B].
tli.min_x(z, LL=0.195, NF=0)[source]¶

Function to minimize for finding \(x(z)\) according to the formula given in append D of [arXiv:1001.4783]. If you use e.g. scipy’s newton method to minimize the resulting funciton, be careful to choose a ‘good’ starting value:

>>> import scipy.optimize as sopt
>>> import scipy.optimize as sopt
>>> sopt.newton(min_x(z = 10), .7, tol=1e-15)
0.5876680790030352
>>> sopt.newton(min_x(z = 12), .7, tol=1e-15)
0.5787751108231223
>>> sopt.newton(min_x(z = 12), .1, tol=1e-15)
RuntimeWarning: Tolerance of 0.00011 reached 
  warnings.warn(msg, RuntimeWarning)
Parameters:
  • z – Input z value.
  • LL – \(\Lambda L\) as in [arXiv:1001.4783].
  • NF – Number of quark flavors.
Returns:

Function. The value that minimizes that function is the sought-after value for \(x\).
tli.pretty_print(val, err, extra_err_digits=1)[source]¶

tli.tli(obs, args, Lrange, n_cut, lrange, x_val, z)[source]¶

Calculate the tree level improvment of an observable. This is accomplished in two steps.

  1. Estimate the continuum limit of obs by performing two fits of the form \(a_0 + a_2 (a/L)^2 + a_3 (a/L)^3 + a_4\, (a/L)^4\) (i.e. we assume obs to be \(O(a)\)-improved) to data generated by applying obs to the values in Lrange, one fit using all values in Lrange, and one omitting the first n_cut points. The difference is then used as an estimate for the systematic error.
  2. Calculate \(\delta (a/L)\) for the values of \(L/a\) provided in lrange.

A minimalistic example:

>>> from tli import tli
>>> tli, dtli, cl = tli(lambda L, x, z : 1. + 2./L**2, (), 
...                     range(50,100,2), 10, (10,14), 0, 0)
>>> tli
[0.019999999999999792, 0.010204081632652738]
>>> dtli
[1.0200135891298212e-10, 1.0102175402546249e-10]
>>> cl
1.0000000000000002
Parameters:
  • obs – The observable (function).
  • args – The observable is expected to have a call signature like obs(L, x_val, z, *args), where x is the x-value obtained using min_x, L the number of lattice points in each direction and z the mass parameter.
  • Lrange – A iterable giving the lattice resoultions that should be used in the fit to extract the continuum limit.
  • n_cut – Number of data points to be omitted in a second fit to estimate the systematic error for the continuum limit.
  • lragne – Values for \((L/a)\) for which to calculate the tree level improvement.
  • x_val – Value for \(x\) as in [arXiv:1001.4783].
  • z – Mass parameter.
Returns:

Tuple (tli, d_cli, cl) with tree level improvement coefficients cl, errors on tli coefficients d_cli, and continuum limits cl.

observables – Various tree level observables in the SF.¶

Note that the documentation for now only covers the primary observables contained in this module.

fX(L, T, x0, m, gam, thetah, thetal, gamp=matrix([[ 1.+0.j, 0.+0.j, 0.+0.j, 0.+0.j],
[ 0.+0.j, 1.+0.j, 0.+0.j, 0.+0.j],
[ 0.+0.j, 0.+0.j, -1.+0.j, 0.+0.j],
[ 0.+0.j, 0.+0.j, 0.+0.j, -1.+0.j]]))

Template for tree level heavy light boundary-to-bulk correlation functions of the form

\[f_{\Gamma, \Gamma'}(x_0, \theta_{\mathrm h}, \theta_{\mathrm l}, z) = \frac {a^6} 2 \sum_{\mathbf y, \mathbf z} \langle \psi_{\mathrm h}(x) \Gamma \overline \psi_{\mathrm l}(x) \overline \zeta_{\mathrm h}(\mathbf y) \Gamma' \zeta_{\mathrm l}(\mathbf z) \rangle\]
Parameters:
  • L – Spatial lattice extent.
  • T – Temporal lattice extent.
  • x0 – Position of the first spin matrix insertion.
  • m – Bare mass of heavy quark.
  • gam – First spin matrix insertion.
  • thetah – Phase angle of the heavy quark.
  • thetal – Phase angle of the light quark.
  • gamp – Spin matrix insertion at the boundary.

propagator.Sf(p, t, u, T, m=0.0)[source]¶

Tree level fermion propagator in the Schrödinger functional with an Abelian background field in the time-momentum representation

\[S_{\mathbf p}(t,u) = (D^{(0)} + m)^{-1}\,,\]

where \(D + m\) is the Dirac operator.

Parameters:
  • t – Initial time.
  • u – Final time.
  • p – Spatial momentum. Phase angles must be included here.
  • T – Lattice extent.
  • m – Bare mass.
